# TO-DO-app

### Проект был написал с использованиям сборщика задач Gulp 4

Для работы с данной сборкой необходимо выполнить следующие шаги:

1. Склонируйте все содержимое репозитория **git clone https://github.com/LittleFinder/TO-DO-app.git** или же нажмите на клавишу **CODE**
далее на **DOWNLOAD ZIP** для загрузки проекта на ваш компьютер.

2. Структура проекта имеет такой внешний вид: <br>
   TO-DO-APP-MAIN
    * app
      - css
      - images
      - js
      - scss
      -index.html
    * .gitignore
    * .gulpfile.js
    * package.json
    * package-lock.json
   
Находясь в корне проекта TO-DO-APP-MAIN , запустите команду **npm install**, которая установит все находящиеся в package.json зависимости

3. После этого в браузере откроется окно с TO DO app.
    TO DO app позволяет пользователю : 
    * добавлять задачи, которые необходимо выполнить
    * отмечать задачи как выполненые
    * удалять задачи
    * использовать фильтр задач
    * редактировать задачи
